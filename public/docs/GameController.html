Class: GameController

GameController()

Manages the state of the game. Responsible for loading different game states (changing the UI) and submitting and getting scores.

Constructor

new GameController()

Creates a new instance of a GameController object. Creates an instance of Box and an instance of GameEngine. Also adds an event listener that prevents left clicking (some users were using left click to move the mouse without the box being able to respond).
